# Extracted Requirement

- Build an application that sends happy mails.
- It should have one entity called 'mail' with two fields:
  - 'isHappy'
  - 'mailList'
- It should include two processors:
  - 'sendHappyMail'
  - 'sendGloomyMail'
- It should include two criteria to define if the mail is happy or gloomy.
