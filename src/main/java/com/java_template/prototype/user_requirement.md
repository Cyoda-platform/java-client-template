# Complete Requirement Specification

- **Application Type:** Java-based application

- **Data Source:**  
  - Ingest Nobel laureates data from the **OpenDataSoft API**

- **Core Entities to Model:**  
  - **Job**  
  - **Laureate**  
  - **Subscriber**

- **Functional Requirements:**  
  - Implement a basic **workflow engine** to handle:  
    - Data processing of the Nobel laureates information  
    - Distribution of relevant updates/notifications to subscribers
