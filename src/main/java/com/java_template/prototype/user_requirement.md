# Complete Requirement

- Develop an application that retrieves book data from the Fake REST API (https://fakerestapi.azurewebsites.net/index.html), analyzes key metrics based on the fields: "id", "title", "description", "pageCount", "excerpt", and "publishDate".  
- The application will generate a summary report including insights on book titles, total page counts, and publication dates, along with a brief description and excerpt of popular titles.  
- This report should be emailed to the analytics team weekly.  
- The data extraction should be automated to run every Wednesday.
