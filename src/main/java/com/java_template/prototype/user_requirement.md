# Complete Requirement for 'Purrfect Pets' API App

- **Project Name:** Purrfect Pets API app

- **Functionality:**
  - Ingest data from Petstore API (pets API).
  - Send data to subscribers.

- **Entities:**
  1. **Pet**
  2. **Subscriber**
  3. **Job**

- **Additional Details:**
  - The app should be fun (implying user-friendly, engaging design or features).
  - Use Petstore API data as the source of pets information.
  - Manage the process of ingestion and distribution of pets data to subscribers.
  - Maintain three distinct entities as the core components of the system: pet, subscriber, and job.
