# Requirement: Inventory Reports using SwaggerHub API

- Generate reports on inventory data using the SwaggerHub API at https://app.swaggerhub.com/apis/CGIANNAROS/Test/1.0.0#/developers/searchInventory.

- The application should generate reports summarizing key metrics like:
  - the total number of items,
  - average price,
  - total value,
  - and other statistics, depending on the available data fields.

- Reports should be displayed in a user-friendly format (e.g., table, chart).

- Error handling and clear presentation of data are crucial.
