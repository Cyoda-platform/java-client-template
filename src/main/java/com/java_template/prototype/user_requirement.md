# Extracted Requirement (verbatim)

- An application that uses the Restful Booker API to generate reports on bookings. The application should be able to retrieve all bookings, filter bookings based on criteria like booking dates, total price, or deposit paid status, and generate reports summarizing booking data, including total revenue, average booking price, and the number of bookings within specific date ranges. The reports should be displayed in a user-friendly format (e.g., table, chart).

# Structured Requirements (preserving original wording and order)

- Application purpose
  - An application that uses the Restful Booker API to generate reports on bookings.

- Functional capabilities
  - The application should be able to retrieve all bookings.
  - The application should be able to filter bookings based on criteria like booking dates, total price, or deposit paid status.
  - The application should be able to generate reports summarizing booking data, including:
    - total revenue
    - average booking price
    - the number of bookings within specific date ranges

- Presentation
  - The reports should be displayed in a user-friendly format (e.g., table, chart).
