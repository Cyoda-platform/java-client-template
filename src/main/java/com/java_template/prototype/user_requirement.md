# Purrfect Pets API — Requirements

- Create a Java 21 Spring Boot application called 'Purrfect Pets' API that uses Petstore API sample data as seed.

- Features to include:
  - REST endpoints to list pets, get pet by id, create pet, update pet, delete pet
  - adopt pet endpoint that changes status to 'adopted' and records adopter info
  - search/filter pets by status/type/tags
  - user favorites (simple user model, in-memory or H2)
  - Swagger/OpenAPI documentation
  - DTOs, service and repository layers, entity models, validation, exception handling, and unit/integration tests
  - Dockerfile, README with run instructions, and example requests

- Use Petstore sample data to pre-seed database on startup.

- Project should target Java 21 and Spring Boot, include Maven/Gradle build, and provide clear package structure.

- Keep code organized for extendability.

- Provide Git branch setup and initial commit.
