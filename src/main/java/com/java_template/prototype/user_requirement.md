# Extracted Requirement

- Build an application that tracks user activities from the Fakerest API. The application should include:
  - Data Ingestion: Fetch activity data for users from the API,
  - Data Processing: Analyze the data to identify patterns in user activity (e.g., frequency, types of activities),
  - Reporting: Generate daily reports summarizing user activities, highlighting trends and anomalies,
  - Publishing: Send the daily reports to the admin email.
- The data ingestion process should be scheduled to run every day.
