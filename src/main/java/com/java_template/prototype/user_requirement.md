# Cover Photo Gallery Application

Hello! I’d like to build an application that creates a gallery of cover photos fetched from the Fakerest API - https://fakerestapi.azurewebsites.net/index.html. The application should include:

- Data Ingestion: Retrieve cover photo data from the API.
- Data Display: Organize and display cover photos in a user-friendly gallery format.
- User Interaction: Allow users to click on photos for a larger view and to leave comments.
- Reporting: Generate monthly reports on the most viewed cover photos.
- Notifications: Notify users about new cover photos added to the gallery.

Data ingestion should be scheduled to run weekly to keep the gallery updated.
