# Complete Requirement

- Build a **Java service** that saves Hacker News items.
- The items must be saved in the **JSON format of the Firebase Hacker News API**.
- Provide a **POST endpoint** to:
  - Store the Hacker News item received in the request body.
  - Return an ID for the stored item.
- Provide a **GET endpoint** to:
  - Retrieve a Hacker News item by its ID.
